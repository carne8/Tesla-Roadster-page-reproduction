# Fable React template

A template to create web applications with F# and React

It use:
  - Fable
  - Elmish
  - React
  - Webpack

## Usage

- Clone this repo

- Install dependencies
    ```bash
    yarn install
    #or
    npm install
    ```
- Start the server
    ```bash
    yarn start
    #or
    npm run start
    ```
- Build the project
    ```bash
    yarn build
    #or
    npm run build
    ```
